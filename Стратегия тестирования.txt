Функциональное тестирование:

Проверка корректности заполнения формы (все поля заполнены, данные введены в соответствии с правилами валидации).
Проверка возможности разворачивания базы данных в разных регионах.
Проверка корректной работы функции по созданию базы данных (в базе данных отображаются все введенные данные, база данных создается без ошибок).

Тестирование производительности:

Проверка скорости создания базы данных при развертывании в разных регионах.
Проверка наличия лагов и задержек при работе с созданной базой данных.

Тестирование безопасности:

Проверка возможности несанкционированного доступа к базе данных.
Проверка наличия мер защиты от SQL-инъекций и других атак на базу данных.

Тестирование совместимости:

Проверка возможности работы базы данных с различными приложениями и ОС.
Проверка совместимости базы данных с другими сервисами Облачного сервиса хранения и обработки информации.

Тестирование восстановления данных:

Проверка возможности восстановления базы данных при сбое системы.
Проверка корректности восстановления данных в базе данных после сбоя.
